# Odoo MRP Batch Scaling

Installable Odoo 18 addon for scaling manufacturing quantities using a batch count and optional machine-size factor.

## Features

- Opt-in batch scaling per manufacturing order
- Batch and machine-size factors
- Automatic finished-product quantity update
- Automatic raw-material quantity update
- Positive-value validation and recursion-safe writes

## Installation

1. Add this repository to the Odoo addons path.
2. Update the Apps list.
3. Install **Portfolio MRP Batch Scaling**.

Module directory: `portfolio_mrp_batch`

## Dependencies

- `mrp`
